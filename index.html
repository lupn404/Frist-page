O que são Tags

As tags são usadas para informar ao navegador a estrutura do site.

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.

As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento.

Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.

Resumo

HTML (Linguagem de Marcação de Hipertexto) é o código que utilizamos para estruturar uma página web e seu conteúdo.

O conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas
